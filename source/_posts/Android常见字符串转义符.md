## android中XML标志符的数字和字符串转义符
```
"   双引号  &#34; 或 &quot;
‘   单引号  &#39; 或 &apos;
&   and符号 &#38; 或 &amp;
<   小于号  &#60; 或 &lt;
>   大于号  &#62; 或 &gt;
@   at符号  &#64;
©   版权    &#169;或 &copy;
®   注册商标 &#174;或 &reg;
™   商标    &#8482;
¥   人民币  &#165;或 &yen;  
```

## 下面字符被定义为空白字符
```
空格 &#x0020;
Tab &#x0009;
回车 &#x000D;
换行 &#x000A;
```
